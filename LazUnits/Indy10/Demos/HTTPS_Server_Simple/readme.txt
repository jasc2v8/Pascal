
To configure this demo before compiling, add Indy10:

	Lazarus Menu: Package, Open lpk: indylaz_runtime, Use, Add to Project
	
To ADD the DEMO certificates to your Windows PC, run from cmd with Admin privs:

	managecerts.cmd -a
	
To REMOVE the DEMO certificates, run from cmd with Admin privs:

	managecerts.cmd -r
