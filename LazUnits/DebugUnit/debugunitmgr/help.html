LazDebugUnit

Overview

DebugUnitMgr adds or removes a DebugForm to a Lazarus Project.

The DebugForm consists of the units debugform.pas and debugform.lfm.

Currently, debugging support for Free Pacal / Lasarus IDE has limited support on Windows.
In lieu of a fully functional debugger, the user  can debug with "old school" methods.

Examples:
Debugln('Entering Function Add...);
Result:=Add(2,2);
Debugln('Exit Function Add, check for exceptions, result should be 4, actual is %n ', Result);
